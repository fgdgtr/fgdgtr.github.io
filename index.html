Comment utiliser le Rack
📱 Étape 1 : Scannez le QR Code

Où est le QR ? Il est collé sur le montant du RACK ATTENTE.

Comment faire :

1️⃣ Ouvrez l'appareil photo de votre téléphone

2️⃣ Pointez vers le carré noir/blanc du QR code

3️⃣ Une notification apparaît → cliquez dessus

4️⃣ Vous arrivez sur cette application ! ✅

📍 Étape 2 : Choisissez votre emplacement

Vous voyez le RACK ENTIER :

🔴 Haut (D) → D1 à D6

🟠 Milieu-haut (C) → C1 à C6

🟡 Milieu (B) → B1 à B6

🟠 Bas (A) → A1 à A6

Action : Cliquez sur l'emplacement où vous mettez votre colis

✏️ Étape 3 : Remplissez les informations

🔴 N° CLIENT — obligatoire, sauf si Hors production

Exemple : 12345, CLIENT001…

🟠 Entreprise, ARC, AFS

Plusieurs AFS ? Séparez par virgules ou espaces

📊 Couleurs automatiques selon l'âge :

🟢 0-15j · 🔵 15-21j · 🟡 21-28j · 🔴 +28j

⏱ Compte à rebours 90 jours

Chaque colis a un délai maximal de 90 jours.

En bas de chaque colis, vous verrez combien de jours restent avant les 90 jours.

Exemple :

• Si 60 jours ont passé → "30 jours restants avant 90j"

• Si 85 jours ont passé → "5 jours restants ⚠️"

🔴 Les Hors production n'ont pas de compte à rebours

🎉 Vous maîtrisez le RACK !

📱 1. Scanner le QR code au rack

📍 2. Cliquer sur votre emplacement

✏️ 3. Entrer le N° Client + infos

✓ 4. Cliquer "+ Ajouter ce colis"

💡 Chaque colis affiche ses jours restants avant 90j

• Modifier un colis → ✏️ · Supprimer → ✕

• Hors prod. → toggle violet, N° client optionnel
